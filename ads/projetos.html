Estratégias de monitoramento de dados com plataforma open-source Arduino

Descrição: Na indústria, o monitoramento de dados, providos de microcontroladores e sensores de um sistema de automação deve ser de claro entendimento, com informações intuitivas, compreensíveis, seguras e objetivas. Estas informações podem ser meros números, texto, ou gráficos de forma a apresentar o estado atual de leitura do comportamento de tal sistema. As informações do estado atual, em conjunto com os dados de tempo anteriores, é possível prover uma tendência futura do comportamento do sistema, e assim, tomar decisões que podem ser antecipadas. Atualmente, uma plataforma open-source de hardware amplamente utilizada chamada Arduino, combinado com linguagens de programação com excelente recursos de capacidade gráfica para monitoramento, como por exemplo, Processing, é possível desenvolver um framework que conecte essas duas ferramentas, para proporcionar informações de monitoramento intuitivos e de forma gráfica. Além do mais, pretende-se desenvolver uma biblioteca para estudo de caso de monitoramento de sensores comuns. Esta tecnologia open-source aliada com ferramentas gráficas poderá futuramente ter sua aplicação expandida para sistemas de monitoramento e controle na industria pelo seu baixo custo, texto, ou gráficos de forma a apresentar o estado atual de leitura do comportamento de tal sistema.
